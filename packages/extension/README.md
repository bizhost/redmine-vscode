# Redmine Issues

VS Code에서 Redmine 일감을 관리하는 확장.

## 기능

- **사이드바**: 내 일감(프로젝트별 그룹) / 프로젝트 전체 일감, 50개 단위 페이징
- **검색**: pane 상단 검색바 토글(🔍) — 제목/`#번호`, 닫힌 일감 포함
- **일감 상세** (탭별로 열림): 제목/설명/상태/우선순위/담당자/범주/일정/진척도 인라인 편집
- **댓글**: 작성(비공개 지원), 파일 첨부, 이미지 붙여넣기
- **첨부파일**: 이미지 인라인 미리보기, 클릭 시 브라우저 다운로드
- **새 일감 만들기**: 프로젝트별 유형/담당자/범주 동적 로드
- **일감 다운로드**: 목록 다중 선택(Ctrl+클릭) → 우클릭 → `<번호>/issue.md` + `attachments/`
- **파일 연관 일감**: 탐색기 우클릭 → 커밋 메시지 `#번호` 기반 관련 일감 조회
- 상위/하위/연결된 일감 링크

## 시작하기

1. 설정에서 `redmine.url` 입력 (예: `https://redmine.example.com`)
2. 명령 팔레트 → `Redmine: Set API Key` → 본인 API key 입력
   - 발급: Redmine → 내 계정 → 우측 "API 접근키"
   - key는 VS Code SecretStorage에 저장됨 (파일에 남지 않음)
3. 활동바 Redmine 아이콘 클릭

## 설정

| 설정 | 설명 |
|---|---|
| `redmine.url` | Redmine 서버 URL (필수) |
| `redmine.projectIdentifier` | '내 일감' 범위를 특정 프로젝트로 한정 (선택) |
| `redmine.downloadPath` | 일감 다운로드 기본 경로. 절대/`~`/상대(워크스페이스 기준). 비우면 매번 선택 |

## 요구사항

- Redmine REST API 활성화 (관리 → 설정 → API)
- 본인 계정의 API 접근키
